# Semantic Search using Endee Vector Database

## Project Overview

This project demonstrates semantic search using Endee vector database.
It finds meaning in text, not just keywords.

## Features

- Convert text to embeddings using AI
- Store and search using vector similarity
- Find relevant results based on meaning

## Tech Stack

- Python 3.11
- Sentence Transformers
- NumPy
- Endee Vector Database

## Setup Instructions

1. Clone this repository
2. Install dependencies:
   pip install sentence-transformers numpy
3. Run the project:
   python search.py

## How it Works

1. Documents are converted to embeddings
2. User query is converted to embedding
3. Similarity is calculated
4. Most relevant results are returned

## Vector Database

This project uses Endee as the vector database.
GitHub: https://github.com/endee-io/endee
